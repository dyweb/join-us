## 东岳是谁?

我们是东岳网络工作室 (Dongyue Studio), 交大校内一个默默无闻的学生组织, 随缘结友, 技术相长. 我们期望有不定期分享, 有一对一指导, 辅以校内外实战项目, 在技术、设计、产品、运营等各方面让自己有所成长, 让组织有所积累, 让世界有所改变.
